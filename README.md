# Scientific Calculator

A Python GUI scientific calculator built with Tkinter.

## Features

- Standard operations: +, -, *, /
- Scientific functions: sin, cos, tan, log, sqrt, exp, pi
- Input history
- Fully clickable buttons GUI
- Number buttons and function buttons color-coded
- Hover effects for buttons
- Clean, responsive layout

## How to Run

1. Install Python 3
2. Run the program:

```bash
python main.py
